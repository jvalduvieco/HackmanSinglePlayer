HackmanSinglePlayer

HackmanJS is an obvious clone of pac-man implemented in JavaScript using MelonJS engine. This is a fork of amamaenko's work I evolved to near complete functionality and updated to melonJS current (0.9.7).

You can find a playable here.

There is also a Hackman Multiplayer. You can find client here and Erlang/OTP server here.
